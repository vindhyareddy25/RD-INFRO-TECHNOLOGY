\# Customer Churn Prediction



\## Overview

Customer churn prediction is a machine learning project that predicts whether a customer is likely to leave a telecom service. This helps businesses improve customer retention strategies.



\## Problem Statement

Customer churn is a major issue for subscription-based businesses. Losing customers affects revenue and business growth. This project helps identify customers who are at risk of churning.



\## Objective

Build a machine learning model to predict customer churn using customer account and service details.



\## Dataset

\- Dataset: Telco Customer Churn Dataset

\- Source: Kaggle

\- Records: 7043 rows

\- Features: 21 columns
